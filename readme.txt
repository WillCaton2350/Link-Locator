URL ENUMERATION TOOL

Bio:

    Locates various subdomains and subdirectories from an allowed total of 3 websites, and outputs them to separate CSV files.

Dependencies:

    Python 3.12.0
    requests==3.3.2
    beautifulsoup4==4.12.3

Setup:

    Install VSCODE IDE
    Install Python 3.12.0 interpreter
    Install VSCODE's Python Extension
    Download zip file or git clone the repo

Usage:

    Specify 3 URLs in the configurations data.py file
    Run the terminal command: "Python3 main.py"
    Check the current directory for outputted CSV Files.
